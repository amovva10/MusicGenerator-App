# MusicGenerator-App

This web application generates music recommendations based on your current mood and preferred language. It uses a short questionnaire to gather information about your mood and language preferences.

## How It Works

1. Answer a few quick questions about your mood and preferred language.
2. Click "Generate" to get a random music recommendation.
3. Click "Generate again" to get more songs from that category 

## Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL
